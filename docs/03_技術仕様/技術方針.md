## スタック（決定案）

| 層                 | 選定                                                                             | 理由                                                                            |
| ------------------ | -------------------------------------------------------------------------------- | ------------------------------------------------------------------------------- |
| フレームワーク     | **Next.js**（App Router）                                                        | 第4回と同系。ルーティング・ビルドが揃っている                                   |
| UI                 | **Tailwind CSS** + **shadcn/ui**（段階導入）                                     | `SHADCN_UI_RULES` §13 · 現状は表 UI 中心にカスタム CSS（`REACTFLOW_RULES` §0）  |
| 図                 | **@xyflow/react**（React Flow）                                                  | カスタムノード・エッジ・エクスポートの実績                                      |
| 言語               | **TypeScript**                                                                   | ドメイン型（NodeIR）を厳密に                                                    |
| レイアウト         | **自前**（dagre はノード配置に使わない）                                         | 9列+: 段→tier · 列→level（ADR-012）· 8列: 行順+Level · **10列推奨**             |
| 状態               | React state + **localStorage**（下書き）+ **Supabase**（クラウド正本 · ADR-013） | localStorage キー `flowchart-studio:draft-v1`（SSOT: `browser/storageKeys.ts`） |
| 認証・DB           | **Supabase Auth** + Postgres（`flow_documents` · `jsonb`）                       | ADR-013 · 2026-05-27                                                            |
| ホスト             | **Vercel**（本番 · Preview 分離予定）                                            | `flowchart-studio-dun.vercel.app`（`-dun` 必須）                                |
| テスト（Phase 1+） | Vitest（ドメイン層）                                                             | `lib/flowchart/**/*.test.ts`                                                    |

**描画エンジン比較（[ADR-010](<../03_技術仕様/意思決定記録(ADR).md>)）:** 本線は React Flow 版（`flowchart-studio`）。Mermaid プレビュー版は **Next.js**（`flowchart-web-mermaid` · dev ポート 3001）。Vite 単体プロトタイプは**採用しない**。

| `model/document.ts`           | JSON シリアライズ · 正規化                 | DL は client から呼ぶ   |
| `visual/flowColors.ts`        | ノード色 · 枠 · 凡例定数                   | キャンバス SSOT         |

| モジュール                      | 配置                                                |
| ------------------------------- | --------------------------------------------------- |
| `toMermaid.ts`                  | `yk-tool/flowchart-web-mermaid`（比較用 · ADR-010） |

## レイアウト方針（重要）

- **dagre / elk でノード位置を決めない**（Level・行順が上書きされるため）。
- 座標は **自前格子**（[データモデル](./データモデル.md) · `graph/layoutGrid.ts`）:
  - **9 列以上:** 縦 = `tier`（段）· 横 = `level`（列）— 同じ段で横並び
  - **8 列以下:** 縦 = 行インデックス · 横 = `level`
  - 共通: `rowHeight + gapV` · `baseLeft + level * (width + gapH)`
- エッジの**経路**のみ React Flow の `smoothstep` / `step` または自前 orthogonal。
