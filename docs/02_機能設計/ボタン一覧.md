## 機能別整理（提案）

ゾーン（画面上の位置）ではなく **ユーザーが何をしたいか** で束ねる。配置は次節。

| #   | 機能カテゴリ           | 含む操作（現 ID）                    | 頻度・優先度                                  |
| --- | ---------------------- | ------------------------------------ | --------------------------------------------- |
| 1   | **同期**               | H1 · O1 · C3※                        | **最高** — Primary はここだけ                 |
| 2   | **表の編集**           | T1 · R1 · （セル入力）               | 高 — 表直上・行末                             |
| 3   | **取込**               | C1 · C2 · M9                         | 高（初回）/ 中 — Excel・貼り付け・装置 bundle |
| 4   | **書き出し**           | H2 · T2 · T3 · M6 · M7               | 中 — ファイル DL · クリップボード · 画像      |
| 5a  | **データの置換**       | H3 · M9                              | 低〜中 — 破壊的 · 確認あり                    |
| 5b  | **雛形・例**           | M1〜M5 · F1 · F2                     | 低（学習・デモ）                              |
| 6   | **ナビ・レイアウト**   | N1〜N7 · V1 · V2                     | 常時 — 選択・折りたたみ                       |
| 7   | **表示（プレビュー）** | P1〜P3 · O1                          | 配置ゾーン — ズーム・stale 回復               |
| 8   | **アカウント**         | G1 · L1〜L4 · X1                     | 画面単位                                      |
| 9   | **管理・破壊**         | N3〜N7 · M10 · M11 · A1〜A3 · D1〜D5 | 低 · 確認必須                                 |

※ C3 は **廃止候補**（H1 と重複）。O1 は stale 時のみの補助導線。

### オーバーフロー「その他」のセクション案

メニュー見出しと機能カテゴリを 1:1 にする。

| セクション見出し | 項目（統一名称）                                                                           |
| ---------------- | ------------------------------------------------------------------------------------------ |
| **取込**         | 表に貼り付け · Excelから取込… · import.jsonを取込…                                         |
| **書き出し**     | JSONをダウンロード · 表をコピー · ヘッダーをコピー · PNGをダウンロード · SVGをダウンロード |
| **読込**         | JSONから読込…                                                                              |
| **雛形・例**     | 雛形: はじめから · 雛形: 直線フロー · 例: カレーの作り方 · … · 例を適用 · 例の表示を終了   |
| **オフライン**   | オフラインに保存 / オフライン保存を解除                                                    |
| **危険**         | フローをリセット… · 下書きを削除（非 workspace）                                           |

装置・ユニット・動作の削除（N3〜N7）は **ナビ上に残す**（対象と同じ場所）。

### 配置案（編集者 · モジュール選択済）

Web 調査・レビュー反映後の推奨。

| 場所                 | 常時表示                                      | 備考                                          |
| -------------------- | --------------------------------------------- | --------------------------------------------- |
| **表ペインヘッダー** | **再生成** · **その他** ▼                     | Primary 1 + overflow トリガー（2 個）         |
| **表直上**           | **行を追加** + **取込パネル**（折りたたみ可） | 7月デモ: Excel 導線を見失わないためパネル残置 |
| **プレビュー右上**   | ズーム − / + / ホーム                         | chrome 方針（別トラック）                     |
| **ナビ**             | 開閉 · 選択 · 削除 icon                       | 変更なし                                      |
| **モバイル**         | 表 \| 図 タブ                                 | 変更なし                                      |

**閲覧者:** 再生成は **非表示**（disabled 常時表示は避ける）。**その他** から PNG/SVG のみ。stale 時オーバーレイはリンクなし文言に分岐（余力時）。

**重複整理:** C3 廃止 · 成功メッセージは「再生成」へ誘導（「反映」文言は貼り付け側と分離）。

## 統一名称（現 → 新）

### 命名ルール

| ルール      | 内容                                                                                                           |
| ----------- | -------------------------------------------------------------------------------------------------------------- |
| 語尾        | **動詞形の体言止め**（MNTSQ / 業務 SaaS 慣行）— 「行を追加」「表をコピー」。確認ダイアログの確定のみ「〜する」 |
| 長さ        | 原則 **3 語以内**                                                                                              |
| 助詞        | 対象を取る操作は **を**（行を削除 · 表をコピー）                                                               |
| 取込 / 読込 | **取込**＝外部データを表へ載せる · **読込**＝ファイルで表＋プレビューを丸ごと置換                              |
| 出力        | ファイル保存は **ダウンロード** に統一（JSON · PNG · SVG）                                                     |
| 破壊的操作  | トリガーに **…**（ellipsis）· 確定は「削除する」「リセットする」等                                             |
| 表示        | トリガー **その他** · `aria-label` **その他の操作**                                                            |

## レビュー要約（サブエージェント · 2026-06-25）

**総合: B+** — 方向性は妥当。以下 3 点を実装前に確定推奨。

| 優先 | 指摘                                   | 本ドキュメントでの反映                     |
| ---- | -------------------------------------- | ------------------------------------------ |
| 1    | 7月デモで Excel 取込の discoverability | 取込パネルを表直上に残置 · overflow は補助 |
| 2    | 「反映」の二義性（貼り付け vs 再生成） | C1 → **表に貼り付け** · 用語集で分離       |
| 3    | 閲覧者の disabled Primary              | **再生成は非表示** に変更案                |

その他: 5 を **置換 / 雛形・例** に分割 · **行を追加は表直上** · 出力動詞は **ダウンロード** に統一 · T3 はラベルと中身を一致させる。
