## 用語一覧

| 用語                   | 意味                                                             |
| ---------------------- | ---------------------------------------------------------------- |
| **装置**               | 1 製品ライン・1 機械。社内番号（`internal_code`）で識別          |
| **ユニット**           | 装置内の機能ブロック（例: 供給ユニット · プレスユニット）        |
| **動作（モジュール）** | ユニット内の 1 本のフロー。ナビで選択する単位                    |
| **表**                 | フローの正本（SoR）。行 = ノード、列 = ID · 図形種別 · 接続先 等 |
| **プレビュー**         | 表から自動生成した図（閲覧専用 · 再生成で同期）                  |
| **再生成**             | 表 → レイアウト → React Flow への反映                            |
| **stale**              | 表変更後 · 未再生成でプレビューが古い状態                        |
| **import.json**        | Python 正規化の出力。装置一括取込用                              |
| **legacy_key**         | 旧 localStorage キー解決用（`modules.legacy_key`）               |
