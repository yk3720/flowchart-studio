# 列の意味 — フローチャート表

**図形・色の使い分け:** `yk-memo/00.ai-driven-school/個人テーマ_フローチャートアプリ/02_spec/フローチャート記述ルール.md`

## 9 列（段 + 列 · 推奨 · ADR-012）

Excel の **行 / 列**（または **段 / 列**）付き表向け。M002 サンプル等。

| 列         | 説明                                                                                 |
| ---------- | ------------------------------------------------------------------------------------ |
| ID         | ノード番号（10, 20…）。他行の接続先にも使う                                          |
| 図形種別   | 端子=開始/終了 · 処理=自動ステップ · 判断=分岐 · 入出力=PLC/上位 · 手動入力=オペ操作 |
| 接続先(下) | 下方向の矢印の行き先 ID                                                              |
| 接続先(右) | 右方向の矢印の行き先 ID（判断の No）                                                 |
| **段**     | 縦位置（同じ段 = 同じ高さで横並び）                                                  |
| **列**     | 横位置（0=左、1=右 … 分岐の並び）                                                    |
| Text1      | 図形に表示する主テキスト（例: MR100 · 取付経路A）                                    |
| Text2      | 補足（2行目）                                                                        |
| Text3      | 補足（3行目）                                                                        |

## 10 列（段 + 列 + 色 · 新規雛形の既定）

| 列  | 説明                                                                   |
| --- | ---------------------------------------------------------------------- |
| 色  | 背景色。通常=自動運転 · 黄=重要な判断 · 橙=要注意/異常系 · 青=手動確認 |

schema: `table-10col-v1` · 9列データは後方互換（色=空）

## 8 列（レガシー）

段を表行順に任せる簡易版。Level のみで横位置を指定。

| 列         | 説明                                                                                 |
| ---------- | ------------------------------------------------------------------------------------ |
| ID         | ノード番号                                                                           |
| 図形種別   | 端子=開始/終了 · 処理=自動ステップ · 判断=分岐 · 入出力=PLC/上位 · 手動入力=オペ操作 |
| 接続先(下) | 下方向の行き先 ID                                                                    |
| 接続先(右) | 右方向の行き先 ID                                                                    |
| Level      | 横位置（0=左、1=右）                                                                 |
| Text1〜3   | 図形に表示するテキスト                                                               |

判断を含むフローは **8 列以上必須**（右分岐がない 7 列では No 矢印が出ません）。
