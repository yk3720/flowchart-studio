## ペイン別仕様（確定）

### ペイン 2 — 図

- **ズーム（右上）+ キャンバスのみ**
- 再生成 · 保存 · 文脈 · ステータス等は載せない

### ペイン 3 — 右ドック（デスクトップ）

```
① [表][設計メモ][コメント]  line tabs · 等幅
② 文脈 · ステータス（1行）
③ ツールバー（表タブ時のみ）
④ 表: 横バー + 列ヘッダー（sticky）+ tbody
   設計メモ / コメント: 本文スクロール
```

| 表タブに残す              | 移す                                      |
| ------------------------- | ----------------------------------------- |
| グリッド · ▶ 確認（警告） | CSV/Excel → その他 ▼                      |
|                           | 列の意味 → **デモのみ** その他 ▼ モーダル |
|                           | 回覧 → **コメント** タブ                  |

### その他 ▼（整理後）

| 環境     | 内容                                               |
| -------- | -------------------------------------------------- |
| **本番** | 既存 ADR-018 案から **列の意味を除外**（約12項目） |
| **デモ** | 書き出し + **列の意味（モーダル）** + CSV/Excel    |

### タブ見た目

- デスクトップ: **line tabs**（下線 · pill 廃止 · `fcMobileTab*` 不使用）
- モバイル上段: 図｜表（line tabs へ寄せる · 優先度低）

## ADR-018 追補（#7 · ADR 反映済 · 2026-06-27）

第1弾 ADR-018 の **全幅ヘッダー昇格** · **右タブ2本** · **本番も列の意味をその他** · **未保存バナー** を、本 grill で次のとおり **修正** — 正本は [ADR-018](<../03_技術仕様/意思決定記録(ADR).md#adr-018-3-ペイン入替--右ドック--デモ整備草案--2026-06-27>):

| 項目         | 第一弾 ADR-018              | 本 grill 追補                                 |
| ------------ | --------------------------- | --------------------------------------------- |
| 右タブ       | `[表][設計メモ]` + 表下回覧 | **`[表][設計メモ][コメント]`** · 回覧はタブ化 |
| ヘッダー位置 | Group 外・全幅固定          | **ペイン3内 sticky**（§E 3段 + タブ行）       |
| 未保存       | バナー（本番）              | **D1** バナー廃止（D6 切替確認は維持）        |
| 列の意味     | 本番 · デモともその他       | **デモのみ** モーダル · 本番なし              |
| モバイル     | 図｜表                      | **M1** 表タブ内サブタブ3本 · デフォルト図     |
| タブ UI      | `fcMobileTab*` pill         | デスクトップ **line tabs**                    |

**実装メモ:** `showRightTabs && !authDisabled` は本番のみ右タブ表示（別ターン済 · 未デプロイ）。
