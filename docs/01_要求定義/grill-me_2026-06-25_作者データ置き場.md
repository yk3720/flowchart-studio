## 決定事項

| 論点               | 決定                                                                                                              | 理由                                                                           |
| ------------------ | ----------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------ |
| 作者データの置き場 | リポジトリ直下 **`data/devices/`**（Python 層外）                                                                 | 作者 Excel は Python 専用ではない                                              |
| Git · xlsx         | **`*.xlsx` / `~$*` のみ Git 外**                                                                                  | 手編集 · バイナリ差分回避                                                      |
| Git · import.json  | **コミット対象**（移行期〜当面）                                                                                  | 別 PC でも Web 取込 · seed · E2E が動く                                        |
| 1 装置             | **`import.json` 1 本**                                                                                            | Web 取込 bundle と一致                                                         |
| `_drafts/`         | **今は作らない**                                                                                                  | 担当者別 JSON は将来 `_drafts/{担当者}/` で拡張可                              |
| 正本の優先         | **Excel 正本 · Web 微修正 · 大変更は Excel**                                                                      | 作者の作業場所と整合                                                           |
| 再取込             | **Excel 再取込で Web 微修正は上書き**                                                                             | Excel 正本と整合                                                               |
| 将来の実運         | **会社の装置フォルダ**（社内サーバー）に xlsx + import.json                                                       | 設計データと同居 · Git 外                                                      |
| Web→Excel 段階1    | 表ペインに **「表をコピー」**（TSV · データのみ）と **「ヘッダーをコピー」**（ヘッダー行のみ）— overflow 書き出し | 空白シート入口 + 微修正の Excel 戻し（※当時はサンプル3行案 → 2026-06-25 変更） |
| Web→Excel 段階2    | Python で 1 フロー → xlsx テーブル書き戻し                                                                        | 後回し                                                                         |
| データ移行         | `python/testdata/devices/` → `data/devices/`                                                                      | **完了（2026-06-25 §4）**                                                      |

## 却下した案

| 案                                           | 理由                                      |
| -------------------------------------------- | ----------------------------------------- |
| 作者データを `python/testdata/` に置き続ける | Python 配下に見え作者データの中立性がない |
| リポ外（yk-document のみ）に分離             | プロジェクト内で見失う                    |
| 装置フォルダ丸ごと Git 外                    | import.json も外れ共有できない            |
| TSV ヘッダー付き（データコピー）             | 既存 Excel テーブルへの貼付で 1 行ずれる  |
