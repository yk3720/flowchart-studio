# 列の意味 — フローチャート表

アプリ内の「列の意味（ヘルプ）」と同じ内容。作者向け SSOT。

**図形・色の使い分け（自動機）:** [フローチャート記述ルール.md](./フローチャート記述ルール.md)

## 9 列（段 + 列 · 推奨）

| 列         | 説明                                                                                 |
| ---------- | ------------------------------------------------------------------------------------ |
| ID         | ノード番号（10, 20…）。他行の接続先にも使う                                          |
| 図形種別   | 端子=開始/終了 · 処理=自動ステップ · 判断=分岐 · 入出力=PLC/上位 · 手動入力=オペ操作 |
| 接続先(下) | 下方向の矢印の行き先 ID                                                              |
| 接続先(右) | 右方向の矢印の行き先 ID（判断の No）                                                 |
| **段**     | 縦位置（同じ段 = 同じ高さで横並び）                                                  |
| **列**     | 横位置（0=左、1=右 … 分岐の並び）                                                    |
| Text1      | 図形に表示する主テキスト（例: MR100 · 取付経路A）                                    |
| Text2      | 補足（2行目）                                                                        |
| Text3      | 補足（3行目）                                                                        |

Excel の **行 / 列** 列は **段 / 列** に相当（取込時は列順で読む · ヘッダー行はスキップ）。

## 10 列（段 + 列 + 色 · 推奨）

| 列  | 説明                                                                                           |
| --- | ---------------------------------------------------------------------------------------------- |
| 色  | 背景色。通常=自動運転 · 黄=重要な判断 · 橙=要注意/異常系 · 青=手動確認（表の警告行の色とは別） |

schema: `table-10col-v1` · コード: `TABLE_HEADERS_10` · `flowColors.ts`

## 8 列（レガシー）

| 列       | 説明                             |
| -------- | -------------------------------- |
| Level    | 同じ行内の横位置（0=左、1=右）   |
| Text1〜3 | 図形に表示するテキスト（複数行） |

コード側: `flowchart-studio/specs/03_技術仕様/列の意味.md` · `lib/flowchart/tableColumns.ts`
