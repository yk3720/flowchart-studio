## 1. 目的（1行）

表の **10列目「色」** でノードの**背景色（＋必要なら枠の補助）**を作者が指定し、図上で判断・異常・手動確認を**形＋ラベル＋色**で伝える（色だけに依存しない · 色覚配慮）。

## 2. データ源

| 項目           | 内容                                                                                                            |
| -------------- | --------------------------------------------------------------------------------------------------------------- |
| **正本**       | `FlowchartDocument.table` の **列 index 9**（0-based · 10列目）                                                 |
| **セル値**     | 日本語キーワード **4種 + 空**（下表）。大小文字・前後空白は trim。未知値は **通常（空）扱い + 警告**            |

## 3. セル値 → 意味（作者向け）

| セル値 | 意味             | 図での役割                                                                                            |
| ------ | ---------------- | ----------------------------------------------------------------------------------------------------- |
| （空） | **通常**         | 白背景 · 黒系枠（現状どおり）                                                                         |
| `黄`   | **判断強調**     | 重要な分岐・インターロック条件（[記述ルール §3](./フローチャート記述ルール.md#3-色列--使い分け凡例)） |
| `橙`   | **異常・要注意** | 異常復帰・要注意工程（**表バリデーション警告行とは別レイヤー**）                                      |
| `青`   | **手動・確認**   | オペ確認・手動介入（**手動入力**図形と併用推奨）                                                      |

**やらない（本仕様）**

- 自由入力の HEX / RGB（作者の typo 防止）
- 10列目で **図形種別** を上書き
- エッジ（矢印）色の列制御 — 引き続き `flowColors.ts` の `FLOW_EDGE_STROKE`

---

## 4. 表示ルール（図 · UI）

### 4-1 ノード描画（`FlowShapeNode`）

| colorHint | 背景（fill / `bg-*`）                 | 枠（stroke / border）            | テキスト         |
| --------- | ------------------------------------- | -------------------------------- | ---------------- |
| 通常      | `#ffffff`                             | `#1a1a1a` · **2px**（現行 SSOT） | `text-slate-800` |
| 黄        | `#fef9c3`（tailwind yellow-100 相当） | 同上                             | 同上             |
| 橙        | `#ffedd5`（orange-100 相当）          | 同上                             | 同上             |
| 青        | `#dbeafe`（blue-100 相当）            | 同上                             | 同上             |

- **菱形:** SVG `polygon` の `fill` を上表に差し替え。`stroke` は `FLOW_NODE_FRAME_STROKE` 維持（計画 A 済みの黒枠を壊さない）。
- **矩形系:** `borderColor` / `backgroundColor` を `style` で上書き。
- **端子:** 形状は小さいが **同じ colorHint ルール**（背景のみ変える）。

### 4-2 表 UI

| 項目     | 方針                                                                                            |
| -------- | ----------------------------------------------------------------------------------------------- |
| ヘッダー | `色`（`TABLE_HEADERS_10`）                                                                      |
| 入力     | **セレクト**（空 · 黄 · 橙 · 青）— 自由テキスト不可                                             |
| 列ヘルプ | `COLUMN_HELP_10.色` — `COLOR_COLUMN_HELP`（[記述ルール](./フローチャート記述ルール.md) と同期） |
| エラー行 | 既存の **赤系 ring**（`validateTable` エラー）— **変更しない**                                  |
| 警告行   | 既存の **amber バナー / ring**（`validateTableWarnings`）— **変更しない**                       |

### 4-3 凡例（プレビュー付近）

計画 B どおり **固定表示**（折りたたみ可 · 初版は常時表示でよい）。

```
□ 通常  □ 判断強調（黄）  □ 要注意（橙）  □ 手動確認（青）
```

色見本は実際の fill 色と一致させる。

## 5. `flowColors.ts` との関係

### 現状（2026-06-06）

```text
flowColors.ts
├── FLOW_EDGE_*          … エッジ・Yes/No ラベル（色列の影響外）
└── FLOW_NODE_FRAME_*    … 全ノード共通の枠色・太さ（2px SSOT）
```

ノード背景は **ハードコード**（`#ffffff` / `bg-white`）。`FlowNodeData` に色フィールドなし。

### 実装時の分割（提案）

| 定数 / 関数                         | 役割                | 色列との関係                          |
| ----------------------------------- | ------------------- | ------------------------------------- |
| `FLOW_NODE_FRAME_STROKE` / `WIDTH`  | 枠 SSOT             | **維持** — colorHint でも上書きしない |
| `FLOW_EDGE_STROKE` 等               | 矢印                | **維持**                              |
| **新規** `FLOW_NODE_FILL_BY_HINT`   | hint → 背景 hex     | **色列の正本マップ**                  |
| **新規** `normalizeColorHint(cell)` | 表セル → hint enum  | parse 層                              |
| **新規** `nodeFillStyle(hint)`      | React 用 style 断片 | FlowShapeNode から呼ぶ                |

`flowColors.ts` 先頭コメントの「計画 B で表「色」列から上書き予定」は、**ノード fill のみ**を指す（枠・エッジは固定のまま）。

### データパイプライン（実装タスク用 · 未着手）

```text
table[row][9]
  → parseTable → FlowNode.colorHint?
  → layoutGrid / toReactFlow → FlowNodeData.colorHint?
  → FlowShapeNode → fill / bg 適用
```
