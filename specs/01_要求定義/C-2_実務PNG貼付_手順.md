## ルート A — 実務データ（推奨）

### 1. モジュールを選ぶ

1. 左 **ナビ**（Phase 3 デモ: 装置「プレス機 A」→ ユニット → モジュール）で、今回扱う **1 モジュール** をクリックする。
2. 中央に表エリア、右にプレビューが出る。未選択のときは「モジュールを選択してください」と表示される。

> 実案件に合わせる場合: そのモジュールに相当する **1 本のフロー分** の表だけを取り込む。

### 2. 表を取り込む（いずれか）

**A-1 Excel ファイル**

1. 左ペイン「表」→ **CSV / Excel 取込** 欄の **Excel ファイル…** をクリック。
2. 8列表の `.xlsx` を選択。
3. ステータスに「Excel（シート: …）」と行数が出る → 続けて手順 3 へ。

**A-2 スプレッドシート貼り付け**

1. Excel / Google スプレッドシートで表範囲をコピー（タブ区切り）。
2. 取込欄のテキストエリアに貼り付け → **表に反映**。
3. 行数メッセージを確認 → 手順 3 へ。

### 3. 意図的に 1 か所編集する

C-2 は「編集 → 再生成」の流れ確認が目的。

- 例: 処理ノードの `Text1` を1語変更、または接続先を確認して問題なければ `Level` のメモ列を1セル修正。
- 編集後、ステータスに **「プレビューは古い」**（琥珀色）が出ることを確認する。

### 4. 再生成

1. ヘッダーの **再生成** をクリック。
2. 右プレビューにフローが表示される。
3. **「プレビューは古い」が消える**こと。エラー赤バナーが出たら、行をクリックして直し、再生成を繰り返す。
4. プレビュー上に **閲覧専用（表を編集 → 再生成）** と出ていることを確認（ドラッグで直すツールではない）。

### 5. PNG 出力

1. **再生成直後**（stale でない状態）で **PNG** をクリック。
2. stale のままだと PNG は無効 — ステータスに「先に再生成」と出る（ADR-008 どおり）。
3. ダウンロードした PNG を画像ビューアで開き、文字が読めることを確認。

### 6. 資料に貼付

1. 手順 0 で決めた **貼付先** に PNG を貼る。
2. 貼付した画面を **スクリーンショット** するか、貼付先ファイルを保存（証跡用）。

### 7. 再生成の記録（陳腐化対策 · 推奨）

貼付だけだと「古い図」になりやすい。同じメモに **次の1行** を残す。

```text
元データ: Flowchart Web · 表 JSON または Excel ○○.xlsx
更新手順: 表を直す → 再生成 → PNG を差し替え
```

## ルート B — 雛形（実データ未準備時 · Web ゲートのみ）

1. モジュールを1つ選択（例: **供給ユニット → 供給動作**）。
2. ヘッダーのサンプルから **雛形: はじめから** または **雛形: 直線** を選ぶ。
3. 表で **1 セル** 編集 → **再生成** → **PNG** → 貼付（ルート A の 3〜7 と同じ）。
4. 完了記録の「データソース」に **雛形（Web 完成確認のみ）** と明記する。

---

## 合格基準（すべて ✅ で C-2 完了）

| #   | 基準                                                                                  |
| --- | ------------------------------------------------------------------------------------- |
| 1   | モジュール選択後、表 → 再生成 → プレビュー表示まで **エラー停止なし**（警告のみは可） |
| 2   | 表編集後に **stale 表示 → 再生成で解消** を自分で確認した                             |
| 3   | stale でない状態で **PNG を保存**できた                                               |
| 4   | PNG を **実際の資料に貼付**した（スクショ or ファイル保存あり）                       |
| 5   | 貼った図について **30 秒で流れを説明**できる（自分用で可）                            |

## よくあるつまずき

| 症状                        | 対処                                                                          |
| --------------------------- | ----------------------------------------------------------------------------- |
| **再生成** が押せない       | 左ナビで **モジュール未選択** → モジュールをクリック                          |
| **PNG** が押せない          | 「プレビューは古い」→ **再生成** 後に再試行                                   |
| Excel 取込でエラー          | 8列形式・ID 重複・接続先 ID を確認。[列の意味.md](../03_技術仕様/列の意味.md) |
| プレビューが空のまま        | 赤エラー一覧の行をクリックして修正                                            |
| `localhost:3000` が開かない | ターミナルで `npm run dev`（`flowchart-studio`）                              |
