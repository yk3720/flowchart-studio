## 解決する課題

業務フローや手順を**フローチャート**で表現したいが、次のような負担がある。

- 図形を1つずつ手で並べるのが手間
- 表形式で構造化したデータから、**一貫したレイアウト**で図にしたい
- **ブラウザだけ**で完結させたい（Excel 必須・Windows 専用 exe は不要）

---

## プロダクトの一文

**表（または同等データ）からフローチャートを自動生成し、ブラウザで確認・保存・画像出力できる Web アプリ。**

---

## やること / やらないこと

| やる                                                                                                           | やらない（本プロジェクト）              |
| -------------------------------------------------------------------------------------------------------------- | --------------------------------------- |
| 表 → ノード IR → 自動レイアウト → 画面表示                                                                     | Excel への図形描画（COM）               |
| ノード種（端子・処理・判断・入出力・手動入力）                                                                 | Office Add-in                           |
| 下/右接続、判断の Yes/No ラベル                                                                                | 第4回「図解管理ワークスペース」との統合 |
| JSON 保存・再読込 · **Supabase クラウド保存**（[ADR-013](<../03_技術仕様/意思決定記録(ADR).md>) · 2026-05-27） | Figma 級の自由描画キャンバス            |
| **Vercel 本番公開** · パスワード/OAuth ログイン（ADR-013）                                                     | デスクトップ exe の置き換えを約束しない |
| PNG / SVG エクスポート · テーマ・サイズプリセット（段階的）                                                    |                                         |
