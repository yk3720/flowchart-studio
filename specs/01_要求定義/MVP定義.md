# MVP 定義 — フローチャート Web アプリ

**対象フェーズ:** Phase 1（MVP）— **本ファイルは MVP 完了時点の SSOT**（2026-05-20 時点で AC-1〜8 達成）。  
**実用拡張:** [ADR-009](<../03_技術仕様/意思決定記録(ADR).md>) · [フェーズ計画](../02_機能設計/フェーズ計画.md) Phase 2 · 現在の機能一覧は [README](../README.md)  
**更新:** 2026-05-24（ADR-009 との関係を追記）

---

## MVP の一文

**表データからフローチャートを自動生成し、ブラウザで表示し、JSON 保存と PNG 出力ができる。**

---

## 受け入れ条件（これが揃えば MVP 完了）

| #    | 条件                                                                | 確認方法                                                                                                                   |
| ---- | ------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| AC-1 | 8列（または7/6列）形式の表を **JSON** で読み込める                  | `fixtures/sample-basic.json`（**MVP 完了時**は JSON のみ。表 UI・CSV は [ADR-009](<../03_技術仕様/意思決定記録(ADR).md>)） |
| AC-2 | 5種ノード（端子・処理・判断・入出力・手動入力）が区別して表示される | 種別ごとに形状が異なる                                                                                                     |
| AC-3 | 下方向・右方向の接続が矢印で表示される                              | 接続先 ID が正しく繋がる                                                                                                   |
| AC-4 | 判断ノードからの分岐に Yes / No ラベルがある                        | 右接続・下接続のラベル                                                                                                     |
| AC-5 | **自動レイアウト**（手動ドラッグのみでは不可）                      | 表変更 → 再生成で位置が決まる                                                                                              |
| AC-6 | 図を **JSON** でダウンロード・読み込みできる                        | リロード後も同じ図が復元                                                                                                   |
| AC-7 | 図を **PNG** でダウンロードできる                                   | ファイルが開ける                                                                                                           |
| AC-8 | 1 画面で「表（または入力）＋プレビュー」が完結する                  | 別ツール不要                                                                                                               |

---

## MVP に含める機能

| 機能                  | 実装の目安                             | 優先                                       |
| --------------------- | -------------------------------------- | ------------------------------------------ |
| 表 → IR 変換          | `parseTable.ts`                        | P0                                         |
| 格子レイアウト        | `layoutGrid.ts`                        | P0                                         |
| エッジ生成            | `buildEdges.ts`                        | P0                                         |
| React Flow キャンバス | カスタムノード 5 種 + ラベル付きエッジ | P0                                         |
| サンプルデータ読込    | 初回起動用 fixture 1 件                | P0                                         |
| 再生成ボタン          | 表変更後にプレビュー更新               | P0                                         |
| JSON export / import  | ファイル DL + ファイル選択             | P0                                         |
| PNG export            | viewport キャプチャ                    | P0                                         |
| テーマ 1 種           | 色・線幅の固定セット                   | **MVP+**（完了条件外）                     |
| エラー表示            | 不正 ID・接続先（生成停止）            | **MVP+** の強化表示。停止自体は MVP に含む |
| localStorage 下書き   | ブラウザ保存                           | **MVP+**（ADR-005 · 完了条件外）           |

## MVP に含めない機能（Phase 1 計画時の境界）

| 機能                       | 理由                | 移す先                                 |
| -------------------------- | ------------------- | -------------------------------------- |
| Excel 読み書き             | スコープ外          | —                                      |
| キャンバス上のドラッグ編集 | 表↔図の双方向が重い | Phase 2                                |
| Undo / 図面クリア          | 編集 UX の一部      | Phase 2                                |
| 雛形・テーマ・SVG 等       | 見た目・入力の拡張  | Phase 2 → **ADR-009 で実装済**（上節） |
| スマート・パレット         | クリック追加 UI     | Phase 2/3（**未**）                    |
| 認証・DB                   | 初版不要            | Phase 3+                               |
| ミニモード・常時最前面     | デスクトップ専用 UX | —                                      |

## 非機能（MVP）

| 項目         | 目標                                     |
| ------------ | ---------------------------------------- |

| オフライン   | 不要（localStorage のみ可）              |
| ノード数     | 〜50 ノードで実用速度（目安）            |
