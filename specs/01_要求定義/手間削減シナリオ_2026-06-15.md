# 手間削減シナリオ（Web · Excel 運用）

> **目的:** 「作る操作の手間削減」を具体的な手順に落とす（PM レビュー 2026-06-15）  
> **正本:** Excel 10列表 · `tools/excel_normalize/` · `flowchart-studio`

---

## 現状の最短フロー（editor）

| #   | 操作                                                                             | 画面           |
| --- | -------------------------------------------------------------------------------- | -------------- |
| 1   | `npm run excel:new-device -- Z00002 装置名` で入力用 xlsx を生成                 | ローカル       |
| 2   | Excel で表を編集 · `npm run excel:normalize`（または Python）で `import.json` 化 | ローカル       |
| 3   | Web を開く → 左ナビで装置を選ぶ                                                  | `/`            |
| 4   | **その他 → import.json を取込…**                                                 | ワークスペース |
| 5   | ナビで **動作** を選ぶ → 表を確認 · 必要なら編集                                 | 表ペイン       |
| 6   | **再生成** → プレビュー確認                                                      | プレビュー     |
| 7   | **その他 → PNG を保存**                                                          | 出力           |

**ステップ数目標（現状）:** 新規装置 約 7 ステップ（うち Web 内 4）  
**次の改善候補:** 取込後の先頭動作自動選択 · Excel 正規化の npm 一本化 · 初回 empty 状態の誘導（→ 2026-06-15 UX 対応済み）

**権限（2026-06-15 確定）:** フロー**編集**は editor 同士で可 · **削除**（動作/装置/ユニット/雛形リセット）は所有者または admin のみ（DB: 016 · HANDOFF §6）
